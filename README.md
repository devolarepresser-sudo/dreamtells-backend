# AMBIENTE DE STAGING - SEGURO PARA DESENVOLVIMENTO

Este é o backend de STAGING do DreamTells.
Você pode modificar à vontade SEM AFETAR a produção.

## ⚠️ REGRA IMPORTANTE
- ✅ **PERMITIDO**: Modificar este diretório (dreamtells-staging)
- 🚫 **PROIBIDO**: Tocar em dreamtells-backend (backend de produção)

## Como usar

### 1. Instalar dependências (primeira vez)
```bash
npm install
```

### 2. Iniciar o servidor
```bash
npm start
```

O servidor vai rodar em: http://localhost:3000

## Diferenças com Produção
- Este backend roda LOCALMENTE para desenvolvimento
- O backend de produção está em Render.com
- Mudanças aqui NÃO afetam usuários da Play Store
